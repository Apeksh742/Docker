# Docker-project

** NOTE: Get one file only and the name of file must be docker-compose.yml only.

Requirements: docker-compose, docker and apache server

Method :

get docker-compose.yml and in same directory run command # docker-compose up .

Alternate :

If earlier method doesn't work properly you can try this one.

get docker-compose-alternate.yml and rename it to docker-compose.yml and in same directory run command # docker-compose up
the container will get set-up and you will be ready to go


NOTE: You can change ports and other information accordingly as per your requirement in yml file.
